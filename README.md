# Cybersecurity & DevSecOps Portfolio

Welcome to the repository for my personal portfolio website! This site is designed to showcase my skills, experience, and achievements as a Security Engineer, Cloud Security Expert, and DevSecOps professional.

## 🛠️ Tech Stack

This project is built from the ground up using a lightweight, dependency-free "Vanilla" stack to ensure optimal performance, fast load times, and fine-grained control over the UI/UX:

*   **HTML5:** Semantic markup for clear structure and accessibility.
*   **CSS3:** Custom styling without the use of heavy external frameworks (like Bootstrap or Tailwind). Includes:
    *   **CSS Variables (Custom Properties):** For maintainable theming and a centralized color palette.
    *   **Modern Layouts:** Heavy use of CSS Grid and Flexbox for a fully responsive design.
    *   **Advanced Animations:** Custom `@keyframes` for the dynamic background (ambient glowing orbs, continuous grid shifts) and engaging UI micro-interactions.
    *   **Glassmorphism:** Leveraging `backdrop-filter` for modern, frosted-glass header effects.
*   **Vanilla JavaScript (ES6+):** Handling all client-side interactivity without external libraries (like jQuery):
    *   **Custom Preloader:** A smooth, character-by-character text reveal sequence when the site initially loads.
    *   **Custom Interactive Cursor:** A dynamic dot and ring cursor that tracks mouse movement and expands when hovering over clickable elements.
    *   **Typewriter Effect:** An automated text-replacement animation highlighting various professional roles.
    *   **Performant Scroll Animations:** Utilizing the native `IntersectionObserver` API for efficient, lag-free "reveal-on-scroll" effects.
    *   **Theme Switching:** Logic for toggling Light/Dark modes and expanding the mobile/sidebar navigation menu.

## ✨ Features

*   **Fully Responsive:** Carefully crafted media queries ensure the site looks great on mobile phones, tablets, and large desktop screens.
*   **Dynamic Backgrounds:** Ambient, animated glowing orbs and a shifting grid background create a modern, tech-forward aesthetic.
*   **Interactive UI:** Custom-styled components including a sliding navigation menu, an experience timeline, and sleek contact forms.
*   **Performance Focused:** Zero external UI/JS dependencies keeps the footprint incredibly minimal.

## 🚀 Getting Started

Because this project relies entirely on Vanilla Web Technologies, there are no complex build steps or package managers (like npm or yarn) required to view it.

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` directly in your favorite web browser, or serve it using a simple local server (like VS Code's Live Server extension).
